# 论文添加流程
* 首先开一个issue，issue名称为论文标题，内容为对文章的评价、疑问等，同时可以留下自己的联系方式方便联系交流
* 找准自己所读论文的分类，进入到分类层级最里层，建立以论文命名的文件夹，然后从里向外逐层添加readme和链接
* 点击Pull requests
* 等待master仓库审核并合并request
